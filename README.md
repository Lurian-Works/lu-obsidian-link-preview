# Link preview cards for Obsidian

#### first test version. <br> 
Works great for me.

#### How To Use:
there are two ways of creating a preview card
1. Over the command palette
    add your link to the note, select it and click ctrl+p to open the command palette.
    search for "convert to link preview card"
        this works with simple links as plain text https://... and also with named links like [this is blender](https://blender.org/)

<img width="540" alt="example codeblock" src="readme-src/Screenshot 2026-05-10 134235.png">


2. Manual create a codeblock with "link-preview" as identifier and add the plain link inside the codeblock.
