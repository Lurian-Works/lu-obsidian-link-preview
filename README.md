# Link preview cards for Obsidian

#### first test version. <br> 
works good but just in reading mode for now. Also without a quick command for now.
I will for sure add edit mode support and a less manual way to add the links in the next days.

#### How To Use:

Just create a codeblock with "link-preview" as identifier and add the link inside the codeblock. that´s it. Here is an example:
